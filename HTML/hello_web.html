1. HTML 구성요소

<시작태그 속성='속성값'> 화면에 출력될 내부 문자 </종료태그 >
태그 tag : < >로 묶인 명령어로 시작태그와 종료태그가 한 세트
요소 element : 시작태그와 종료태그로 이루어진 명령어
속성 : 명령어 구체화하는 역할
속성값 : 속성이 가지는 값

2. HTML 구조

<!DOCTYPE> : 문서유형, 형식을 알려주는 태그
<html> : 문서의 시작과 끝 표시
+ <html lang="en"> 이 페이지가 어느 나라 언어로 되어있는지 의미한다. -> 검색 엔진이 참고
<head> : 문서의 정보
<body> : 화면에 출력해서 보여주는 내용

block & inline

HTML 태그는 블록요소와 인라인요소로 이루어져 있다.

블록요소 block element
  • 내용 길이 상관 없이 요소 자체가 전체
  • 기본 줄바꿈
  • 텍스트, 인라인 요소 포함되어 있음
  • 블록 요소 안에 블록 요소 가능
  • 기본 직사각형 형태 + width, height, margin, padding 등으로 변형을 줘서 사용 가능
