DOM : 자바스크립트가 HTML을 문서의 모든 요소에 엑세스하여 변경 사용 가능

HTML DOM 속성 & 메서드

  • DOM 탐색 메소드 (1) 선택 하나 -> document.querySelctor("css선택자표현식")
  • DOM 탐색 메소드 (2) 선택 여러개 -> document.querySelctorAll("css선택자표현식")
  • 요소로 탐색
    • 요소의 id로 탐색 -> document.getElementByID("요소의 아이디")
    • 요소의 name으로 탐색 (1) 선택 하나 -> document.getElementsByName("요소의 이름")
    • 요소의 name으로 탐색 (2) 선택 여러개 -> document.getElementsByTagName("요소의 태그이름")
    • 요소의 class으로 탐색 -> document.getElementsByClassName("요소의 class이름")
