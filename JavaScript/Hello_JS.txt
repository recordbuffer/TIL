자바스크립트
(1) 변수
var : 호이스팅, 할당, 재할당 가능
let : 할당, 재할당 가능
const : 할당만 가능 / 변할 수 없는 값만 선언

→ ${ } 변수를 출력할 수 있음

(2) 자료형 
→ 문자형, 숫자형, 논리형 등
typeof 연산자로 자료형 알아 낼 수 있음

(3) 응답
- alert( ) : 알림 				ex) alert('삭제되었습니다');
- prompt( ) : 입력받을 수 있는 알림		ex) prompt('이름을 입력하세요');
- confirm( ) : 확인 혹은 취소 받는 알림		ex) confirm('정말 삭제하시겠습니까?');

(4) 형변환
